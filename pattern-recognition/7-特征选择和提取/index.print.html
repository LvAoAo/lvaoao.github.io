Feature Selection and Extraction

特征选择和提取

引言

为什么要降维？

  • 如果将数目很多的测量值不做分析，全部直接用作分类特征，不但耗时，而且会影响到分类的效果， 产生“特征维数灾难”问题。
  • 维数灾难：当维数增加时, 输入空间(input space) 的大小随维数指数级增加, 训练数据占整个数据空间的比例将急剧下降, 这将导致模型的泛化困难。 在高维空间中, 样本数据将变得十分稀疏, 许多的相似性度量在高维都会失效。(相同的空间大小下，能存放的高维数据更少，导致训练样本更稀疏)
  • 在保证一定分类精度的前提下，减少特征维数，即进行“降维”处理，使分类器实现快速、准确和高效的分类。
  • 很多时候，观测或收集到的数据样本虽是高维的，但与学习任务紧密相关的是某个低维“嵌入（embedding）”。

基本概念

特征选择，就是从 \(n\) 个度量值集合 \(\left\{x_1, x_2, \ldots, x_n\right\}\) 中, 按某一准则选取出供分类用的子集, 作为降维 \((m\) 维, \(m<n)\) 的分类特征

特征提取，就是使 \(\left(x_1, x_2, \ldots, x_n\right)\) 通过某种变换， 产生 \(m\) 个特征 \(\left(y_1, y_2, \ldots, y_m\right)(m<n)\), 作为新的分类特征 (或称为二次特征)

目的：都是为了在尽可能保留识别信息的前提下, 降低特征空间的维数, 以达到有效的分类

特征选择

一般来说，特征选择步骤如下，主要包括子集搜索和子集评估

严格定义

给定一个特征集 \(X=\left\{x_i \mid i=1,2, \ldots, N\right\}\), 找到该特征集的子集 \[ Y_M=\left\{x_{i_1}, x_{i_2}, \ldots, x_{i_M}\right\}, M<N \] 使得所定义的准则函数 \(J(Y)\) 最优

注: 该准则函数在理想状态下, 是正确分类的概率， 错误率并不容易计算!

子集搜索

  • 特征选择的最优算法:
    • 穷举
    • 分支定界
  • 特征选择的次优算法:
    • 独立算法
    • 顺序前进法 (Sequential forward selection, SFS)
    • 顺序后退法 (Sequential backward selection, SBS)
    • 增\(l\)减\(r\)法
    • 遗传算法

前向搜索

依次在候选集合中增加相关特征

Optimal feature: \(\quad\{\quad\} \Rightarrow\left\{a_2\right\} \Rightarrow\left\{a_2, a_4\right\} \ldots\).

后向搜索

在候选集合中, 依次去除不相关特征（通过子集评估）

Optimal feature: \(\left\{a_1, a_2, \ldots, a_d\right\} \Rightarrow\left\{a_1, a_2, \ldots, a_{d-1}\right\} \Rightarrow\left\{a_1, a_2, \ldots, a_k, \ldots a_{d-2}\right\} \Rightarrow \ldots\)

这些方法是贪心的策略, 因为是在上一轮的基础上考虑本轮最优，所以不一定得到最优特征组合。

其他子集搜索方法: 《模式识别》张学工, 第三版第七章 《特征选择》

子集评估

分类可区分性判据(Separation Criterion) 用于评估特征子集的类别区分性能力：

  • 基于距离的类可区分性判据
  • 基于概率分布的类可区分性判据
  • 基于熵的类可区分性判据

基于距离的判据

搜索一个特征子集，我们希望 :

  • 样本类内的距离尽可能小
  • 样本类间的距离尽可能大

定义：

  1. 各类样本的均值向量 \(m_i \quad m_i=\frac{1}{N_i} \sum_{x \in D_i} x,(i=1, \ldots, c)\)
  2. 所有各类样本的总平均向量 \(m \quad m=\sum_{i=1}^c P_i m_i\),
  3. 样本总类内离散度矩阵 \(S_w\)

\[ S_w=\sum_{i=1}^c P_i \frac{1}{N_i} \sum_{k=1}^{N_i}\left(x_k^i-m_i\right)\left(x_k^i-m_i\right)^T \] 4. 样本类间离散度矩阵 \(S_b\)

\[ S_b=\sum_{i=1}^c P_i\left(m_i-m\right)\left(m_i-m\right)^T \quad \begin{aligned} &\text { 注: 协方差矩阵的迹 } \\ &\text { 等同于方差 } \end{aligned} \]

这种类间散度矩阵的定义在只有两类的情况下和 \[ S_b=\left(m_1-m_2\right)\left(m_1-m_2\right)^T \]

是等价的

类可区分性判据，以各类特征向量的平均平方距离为判据: \[ J_d(x)=\operatorname{tr}\left(S_w+S_b\right) \] 基于距离的判据 - 还可以定义一系列基于类内、类间距离的判据:

\[ J_1=\operatorname{tr}\left(S_w+S_b\right) \]

还可以定义一系列基于类内、类间距离的判据: \[ \begin{aligned} &J_1=\operatorname{tr}\left(S_w+S_b\right) \\ &J_2=\operatorname{tr}\left(S_w^{-1} S_b\right) \\ &J_3=\ln \frac{\left|S_b\right|}{\left|S_w\right|} \\ &J_4=\frac{\operatorname{tr}\left(S_b\right)}{\operatorname{tr}\left(S_w\right)} \\ &J_5=\frac{\left|S_b-S_w\right|}{\left|S_w\right|} \end{aligned} \]

不同判据计算的数值虽然不同， 但是对于特征的排序是相同的， 结果一致。

不足

很难在理论上建立起与分类错误率的联系；没有考虑样本的分布情况

基于概率分布的判据

上面是不同使用特征子集表征样本的情况下，类条件概率密度的图示

它启发我们可用两类概率密度函数的交叠程度（相似程度）来度量可分性，构造基于类概率的可分性判据

它可以是概率密度函数的乘积、比或差的积分

交叠度

交叠度可用两个分布密度函数之间的距离 \(J_p\) 来度量: \[ J_{\boldsymbol{p}}=\int {g[P\left(x \mid \omega_1\right), P\left(x \mid \omega_2\right)}\left.P_1, P_2\right] dx \]

其中\(P_1=P(\omega_1),P_2=P(\omega_2)\)为先验概率

构造的基于类概率的可分性判据应满足的条件:

  1. \(J_P \geq 0\)
  2. 当两类不重叠时, \(J_P=\max\)
  3. 当两类概率密度完全重合时, \(J_P=0\)
  4. 相对于两个概率密度具有 “对称性”

下面介绍三个判据 \(J_B, J_C, J_D\)

\(J_B\) (Bhattacharyya 判据)

\[ J_B=-\ln \int\left[p\left(\boldsymbol{x} \mid \omega_1\right) p\left(\boldsymbol{x} \mid \omega_2\right)\right]^{\frac{1}{2}} d \boldsymbol{x} \] 直观分析：

  • 当两类的概率密度函数完全重合时, \(J_{\mathrm{B}}=0\);
  • 当两类的概率密度函数完全没有交叠时, \(J_{\mathrm{B}}=\infty\)

它与错误率的上界有直接关系： \[ P(\boldsymbol{e}) \leq\left[P\left(\omega_1\right) P\left(\omega_2\right)\right]^{\frac{1}{2}} \exp \left(-J_B\right) \]

证明思路: 转成后验比较 \[ \begin{aligned} p(e)=&\int_{\mathfrak{R}_1} p\left(\omega_2 \mid x\right) p(\mathrm{x}) d x+\int_{\mathfrak{R}_2} p\left(\omega_1 \mid x\right) p(\mathrm{x}) d x \\ \left[P\left(\omega_1\right) P\left(\omega_2\right)\right]^{\frac{1}{2}} \exp \left(-J_B\right)=&\int_{\mathfrak{R}_1}\left[p\left(\omega_1 \mid x\right) p\left(\omega_2 \mid x\right)\right]^{\frac{1}{2}} p(\mathrm{x}) d x+ \int_{\mathfrak{R}_2}\left[p\left(\omega_1 \mid x\right) p\left(\omega_2 \mid x\right)\right]^{\frac{1}{2}} \boldsymbol{p}(\mathrm{x}) d x \end{aligned} \] 又有 \[ \Re_1上: p\left(\omega_1 \mid x\right)>p\left(\omega_2 \mid x\right) \\ \Re_2上: p\left(\omega_2 \mid x\right)>p\left(\omega_1 \mid x\right) \] 故得证

注意\(\Re_1\)的含义为实际类型为\(\omega_1\)的\(x\)区间

\(J_C\) (Chernoff 判据)

\[ J_C=-\ln \int\left[p\left(\boldsymbol{x} \mid \omega_1\right)\right]^s\left[p\left(\boldsymbol{x} \mid \omega_2\right)\right]^{1-s} d \boldsymbol{x} \quad s \in[0,1] \]

  • 对一切 \(0 \leq s \leq 1, J_C \geq 0\)

  • 对一切 \(0 \leq s \leq 1, J_C=0 \leftrightarrow p\left(x \mid \omega_1\right)=p\left(x \mid \omega_2\right)\)

  • 当 \(x\) 的各分量彼此独立时

    \[ J_C\left(s ; x_1, x_2, \cdots, x_n\right)=\sum_{i=1}^n J_C\left(s ; x_i\right) \]

  • 当 \(x\) 的各分量彼此独立时

\[ J_C\left(s ; x_1, x_2, \cdots, x_k\right) \leq J_C\left(s ; x_1, x_2, \cdots, x_k, x_{k+1}\right)J_C\left(s ; x_1, x_2, \cdots, x_k\right) \leq J_C\left(s ; x_1, x_2, \cdots, x_k, x_{k+1}\right) \]

\(J_D\) (D-判据、散度)

在似然比的基础上定义了以下散度作为可分性度量（相对熵的和）: \[ \begin{aligned} &J_D=I_{12}(\mathrm{x})+I_{21}(\mathrm{x})=\int\left[p\left(\mathrm{x} \mid \omega_1\right)-p\left(\mathrm{x} \mid \omega_2\right)\right] \ln \frac{p\left(\mathrm{x} \mid \omega_1\right)}{p\left(\mathrm{x} \mid \omega_2\right)} d \mathrm{x} \\ &I_{12}(\mathrm{x})=E_1\left[\ln\frac{p\left(\mathrm{x} \mid \omega_1\right)}{p\left(\mathrm{x} \mid \omega_2\right)}\right]=\int p\left(\mathrm{x} \mid \omega_1\right) \ln \frac{p\left(\mathrm{x} \mid \omega_1\right)}{p\left(\mathrm{x} \mid \omega_2\right)} d \mathrm{x} \\ &I_{21}(\mathrm{x})=E_2\left[\ln\frac{p\left(\mathrm{x} \mid \omega_2\right)}{p\left(\mathrm{x} \mid \omega_1\right)}\right]=\int p\left(\mathrm{x} \mid \omega_2\right) \ln \frac{p\left(\mathrm{x} \mid \omega_2\right)}{p\left(\mathrm{x} \mid \omega_1\right)} d \mathrm{x} \end{aligned} \]

关系

当两类均是正态分布且协方差矩阵相等时: \[ J_D=\left(\boldsymbol{\mu}_{\mathbf{1}}-\boldsymbol{\mu}_{\mathbf{2}}\right)^{\boldsymbol{T}} \boldsymbol{\Sigma}^{\mathbf{- 1}}\left(\boldsymbol{\mu}_{\mathbf{1}}-\boldsymbol{\mu}_{\mathbf{2}}\right)=8 J_B \]

基于熵的判据

熵 (Entropy), 以Shannon熵为例: \[ H=-\sum_{i=1}^c P\left(w_i \mid x\right) \log _2 P\left(w_i \mid x\right) \]

  • 样本类别确定: \(P\left(w_i \mid x\right)=1, H=0\);
  • 样本类别不确定: \(P\left(w_i \mid x\right)<1, H>0\);
  • 衡量后验概率的集中程度
  • 熵值越大, 说明样本的类别不确定性越大

香农熵 (Shannon Entropy) : \[ H=-\sum_{i=1}^c P\left(w_i \mid x\right) \log _2 P\left(w_i \mid x\right) \] 平方熵 (Square Entropy) : \[ H=J_C^2=\left[P\left(w_1 \mid x\right), P\left(w_2 \mid x\right), \ldots P\left(w_c \mid x\right)\right]=2\left[1-\sum_{i=1}^c P^2\left(w_i \mid x\right)\right] \] 对特征所有取值的熵积分，得到可分性判据： \[ J_{\mathrm{E}}=\int H(x) p(x) d x \]

熵值可以度量后验概率的分布 (集中程度)，后验概率分布在一定程度上体现分类性能的优劣：后验概率分散性越大，熵值越大，一般分类错误率越高，分类效果越差。

对于一个样本, 如果所有类的后验概率是相同的（熵会很大）, 则分类结果非常不确定，例如:

对于\(x\in Re_i\)

  • if \(\mathbf{P}\left(\boldsymbol{w}_{\boldsymbol{i}} \mid x\right)=\frac{1}{c}, c\) 是类别数量，分类错误率: \(e=1-\frac{1}{c}=\frac{C-1}{C}\)
  • if \(P\left(w_i \mid x\right)=1, P\left(w_j \mid x\right)=0, j \neq i\)，分类错误率: \(e=0\)

当然，以熵值作为判据不一定靠谱：有些时候我们需要给出确定的答案，但是前提是分类正确，如果给了一个“确定的”错误答案，可能会造成很严重的后果；在这种容错率低的问题上，一个确定的错误答案，不如一个不那么确定、转而将问题交给人类去判断的分类器。

特征选择策略

特征选择策略即特征子集搜索和子集评估组合起来的过程

  • 过滤式 (Filter)

    特征选择发生在训练过程之前, 单独利用可分性准则来选择特征

    代表性方法: Relief

  • 包裹式 (Wrapper)

    评估准则与分类的错误率直接相关!

    直接将分类器的性能作为特征选择中的子集评估方法

    代表性方法: LVW（拉斯维加斯算法）

  • 嵌入式 (Embedding)

    通过优化的方式同时实现特征选择与评估特征选择

    特征选择和学习器训练同时嵌入到一个优化过程中, 特征选择在学习器训练过程中完成 (有训练过程)

    例如 \[ \min _\omega \sum_{i=1}^m\left(y_i-\omega^T x_i\right)^2+\lambda\|\omega\|_1 \] 易获得稀疏解，是一种嵌入式特征选择方法

过滤式

特征选择发生在训练过程之前, 单独利用可分性准则来选择特征

代表性方法: Relief

思想

  1. 给定“相关统计量” ，度量特征的重要性
  2. 设置一个阈值t, 如果某一个特征的相关统计量大于阈值t, 那么就将其加入特征子集
  3. 特征子集的重要性等于特征子集相关统计量的和

Relief 是为二分类问题设计的, 其扩展变体 Relief-F [Kononenko, 1994] 能处理多分类问题.

假定数据集 \(D\) 中的样本来自 \(|\mathcal{Y}|\) 个类别. 对示例 \(x_i\), 若它属于第 \(k\) 类 \(k \in\{1,2, \ldots,|\mathcal{Y}|\}\),

则 Relief-F 先在第 \(k\) 类的样本中寻找 \(x_i\) 的最近邻示例 \(\boldsymbol{x}_{i, \mathrm{nh}}\) 并将其作为猜中近邻

然后在第 \(k\) 类之外的每个类中找到一个 \(\boldsymbol{x}_i\) 的最近邻示例作为猜错近邻, 记为 \(x_{i, l, \mathrm{~nm}}(l=1,2, \ldots,|\mathcal{Y}| ; l \neq k)\). 于是, 相关统计量对应于属性 \(j\) 的分量为 \[ \delta^j=\sum_i\left(-\operatorname{diff}\left(x_i^j, x_{i, \mathrm{nh}}^j\right)^2+\sum_{l \neq k}\left(p_l \times \operatorname{diff}\left(x_i^j, x_{i, l, \mathrm{~nm}}^j\right)^2\right)\right), \] 其中 \(p_l\) 为第 \(l\) 类样本在数据集 \(D\) 中所占的比例.

然后再通过对比统计量\(\delta^j\)和阈值\(t\)筛选特征\(j\)

包裹式

直接将分类器的性能作为特征选择中的子集评估方法，评估准则与分类的错误率直接相关。

代表性方法: LVW（拉斯维加斯算法）

思想

  1. 在候选特征集中自由选择特征子集
  2. 在特征子集表示的数据集上，运行学习算法（分类算法等）
  3. 用分类的错误率来评估特征子集的好坏

特点

  • LVW 可以减少特征的维数，并且提高分类的准确率
  • 子集搜索采用随机策略。由于每次都要运行分类器，复杂性高，算法运行速度慢

嵌入式

通过优化的方式同时实现特征子集选择与评估特征子集

特征选择和学习器训练同时嵌入到一个优化过程中, 特征选择在学习器训练过程中完成 (有训练过程)

例如 \[ \min _\omega \sum_{i=1}^m\left(y_i-\omega^T x_i\right)^2+\lambda\|\omega\|_1 \] \(L_1\)范数比\(L_2\)范数更容易获得稀疏解，是一种嵌入式特征选择方法

假设x仅有两个属性，那么w有两个分量𝒘𝟏和𝒘𝟐。那么目标优化的解要在平方误差项与正则化项之间折中，即出现在图中平方误差项等值线与正则化等值线相交处。

从图中看出，采用\(L_1\)范数时交点常出现在坐标轴上， 即产生𝒘𝟏或者𝒘𝟐为0的稀疏解。

主成分分析(Principal Component Analysis, PCA)

主成分分析(Principal Component Analysis, PCA)，将原有众多具有一定相关性的指标重新组合成一组少量互相无关的综合指标。

  • 使得降维后样本的方差尽可能大
  • 使得降维后数据的均方误差尽可能小

最大方差思想

使用较少的数据维度保留住较多的原数据特性

定义

  • 将 \(D\) 维数据集 \(\left\{\mathrm{x}_n\right\}, n=1,2, \ldots, N\) 降为 \(M\) 维, \(M<D\)，首先考虑 \(M=1\), 定义这个空间的投影方向为 \(D\) 维向量 \(\mathbf{u}_1\) ，出于方便且不失一般性, 令 \(\mathbf{u}_1^T \mathbf{u}_1=1\)
  • 每个数据点 \(\mathbf{x}_n\) 在新空间中表示为标量 \(\mathbf{u}_1^T \mathbf{x}_n\)
  • 样本均值在新空间中表示为 \(\mathbf{u}_1^T \overline{\mathrm{x}}\), 其中 \(\overline{\mathbf{x}}=\frac{1}{N} \sum_{n=1}^N \mathbf{x}_n\)
  • 投影后样本方差表示为 \(\frac{1}{N} \sum_{n=1}^N\left\{\mathbf{u}_1^T \mathbf{x}_n-\mathbf{u}_1^T \overline{\mathbf{x}}\right\}^2=\mathbf{u}_1^T \mathbf{S u _ { 1 }}\) （使这个方差最大化），其中原样本方差矩阵 \(\mathbf{S}=\frac{1}{N} \sum_{n=1}^N\left(\mathbf{x}_n-\overline{\mathbf{x}}\right)\left(\mathbf{x}_n-\overline{\mathbf{x}}\right)^T\)

目标：最大化 \(\mathbf{u}_1^T \mathbf{S u}_1\), s.t. \(\mathbf{u}_1^T \mathbf{u}_1=1\)

  1. 利用拉格朗日乘子法 \(\mathbf{u}_1^T \operatorname{Su}_1+\lambda_1\left(1-\mathbf{u}_1^T \mathbf{u}_1\right)\)
  2. 对 \(\mathbf{u}_1\) 求导置零得到 \(\mathrm{Su}_1=\lambda_1 \mathbf{u}_1\)
  3. \(\mathbf{u}_1\)是 \(\mathrm{S}\) 的特征向量，进一步得到 \(\mathbf{u}_1^T \mathbf{S u}_1=\lambda_1\)

\(\mathbf{u}_1\) 是 \(\mathrm{S}\) 最大特征值对应的特征向量时方差取到极大值, 称 \(\mathrm{u}_1\) 为第一主成分

最小均方误差思想

使原数据与降维后的数据(在原空间中的重建)的误差最小

定义一组正交的 \(D\) 维基向量 \(\left\{\mathbf{u}_i\right\}, i=1, \ldots, D\), 满足 \(\mathbf{u}_i^T \mathbf{u}_j=\delta_{i j}\)

这里\(\delta_{ij}\)容易引起误解，直观来说就是\(i=j\)时为1，\(i\neq j\)时为0。和我们通常理解的基向量意义是一致的

当基是完全的, 每个数据点可以表示为基向量的线性组合，相当于进行了坐标变换 \(\mathbf{x}_n=\sum_{i=1}^D \alpha_{n i} \mathbf{u}_i\) \[ \left\{\mathbf{x}_{n 1}, \ldots, \mathbf{x}_{n D}\right\} \stackrel{\left\{\mathbf{u}_i\right\}}{\longrightarrow}\left\{\alpha_{n 1}, \ldots, \alpha_{n D}\right\} \] \(\alpha_{n j}=\mathrm{x}_n^T \mathbf{u}_j\) 或 \(\alpha=A^T \boldsymbol{x}\) 其中 \(\mathrm{A}\) 是由 \(\mu_i\) 构成的变换矩阵

那么 \(\mathbf{x}_n=\sum_{i=1}^D\left(\mathbf{x}_n^T \mathbf{u}_i\right) \mathbf{u}_i\)

在 \(M\) 维变量 \((M<D)\) 生成的空间中对其进行表示 \[ \tilde{\mathbf{x}}_n=\sum_{i=1}^M z_{n i} \mathbf{u}_i+\sum_{i=M+1}^D b_i \mathbf{u}_i \] 目标最小化失真度 \(\quad J=\frac{1}{N} \sum_{n=1}^N\left\|\mathbf{x}_n-\tilde{\mathbf{x}}_n\right\|^2\) ，导数置零得到 \[ z_{n j}=\mathbf{x}_n^T \mathbf{u}_j, j=1, . ., M \\ b_j=\overline{\mathbf{x}}^T \mathbf{u}_j, j=M+1, \ldots, D \] 有 \[ \mathbf{x}_n-\tilde{\mathbf{x}}_n=\sum_{i=M+1}^{D}\left\{\left(\mathbf{x}_n-\overline{\mathbf{x}}\right)^T \mathbf{u}_i\right\} \mathbf{u}_i \\J=\frac{1}{N} \sum_{n=1}^N \sum_{i=M+1}^D\left(\mathbf{x}_n^T \mathbf{u}_i-\overline{\mathbf{x}}^T \mathbf{u}_i\right)^2=\sum_{i=M+1}^D \mathbf{u}_i^T \mathbf{S} \mathbf{u}_i \] 拉格朗日乘子法 \[ \tilde{J}=\sum_{i=M+1}^D \mathbf{u}_i^T \mathbf{S u}_i+\sum_{i=M+1}^D \lambda_i\left(1-\mathbf{u}_i^T \mathbf{u}_i\right) \]

求导置零得到 \(\mathrm{Su}_i=\lambda_i \mathbf{u}_i\),

对应失真度为 \(J=\sum_{i=M+1}^D\lambda_i\)

$ J$ 最小时取 \(D-M\) 个最小的特征值， 主子空间对应 \(M\) 个最大特征值

这两种思想在结果上是等价的，都要求样本方差矩阵的最大M个特征向量

算法流程

  • 计算给定样本 \(\left\{\mathrm{x}_n\right\}, n=1,2, \ldots, N\) 的均值 \(\bar{x}\) 和协方差矩阵 \(\mathbf{S}\);
  • 计算 \(\mathrm{S}\) 的特征向量与特征值;
  • 将特征值从大到小排列, 前 \(M\) 个特征值 \(\lambda_1, \ldots, \lambda_M\) 所对应的特征向量 \(\mathbf{u}_1, \ldots, \mathbf{u}_M\) 构成投影矩阵。

利用 PCA处理高维数据

在实际应用中, 样本维数可能很高, 远大于样本的个数在人脸识别中, 1000 张人脸图像, 每张图像 \(100 \times 100\) 像素

  • \(D\) 维空间, \(N\) 个样本点, \(\mathrm{D}>\mathrm{N}\)
  • \(\mathrm{X}\) 是 \(\boldsymbol{D} \times N\) 维的数据矩阵, 其列向量为 \(\left(x_n-\bar{x}\right)\) \(\mathrm{S}=\frac{1}{N} \sum_{n=1}^N\left(\mathbf{x}_n-\overline{\mathbf{x}}\right)\left(\mathbf{x}_n-\overline{\mathbf{x}}\right)^T\) 可以写为 \(\mathbf{S}=\frac{1}{N} \boldsymbol{X} \boldsymbol{X}^T\)
  • S维数? \(D \times D\) 维。考察样本集构成的另一个矩阵 (内积矩阵) \(R=\frac{1}{N} X^T X\), 维数 \(N \times N\) 。通常 (不妨假设) \(\mathrm{N}<<\mathrm{D}\) 。

在实际应用中, 样本维数可能很高, 远大于样本的个数

对 \(R=\frac{1}{N} X^T X\) 求的特征值 \(\lambda_i\) 和特征向量 \(\mathrm{v}_i\) \[ X^T X v_i=\lambda_i v_i \] \(X X^T X v_i=\lambda_i X v_i\)

\(S X v_i=\lambda_i X v_i\) 记: \(\mu_i=X v_i\), 则 \(S \mu_i=\lambda_i \mu_i\)

因此, \(R\) 与 \(S\) 具有相同本征值, 本征向量: \(\mu_i=X v_i\)

调整尺度 \(\mu_i=\frac{1}{\sqrt{\lambda_i}} X v_i \quad\) 满足: \(\left\|\mathrm{u}_i\right\|=1\)

通过这种转换，我们可以求一个更小的矩阵R的特征向量，进而可以根据特征值求得维度更小的特征子集

注：也可以借鉴SVD奇异值分解的思想，对X进行SVD，其中一个奇异矩阵是S的特征矩阵，另一个是R的特征矩阵。而奇异值矩阵\(\Sigma\)对角线上的奇异值和S、R的特征值意义接近。借鉴更简便的SVD方法求出奇异值，从而也可以对矩阵X进行主成分分析。

  • PCA优点
    • 具有很高普适性，最大程度地保持了原有数据的信息；
    • 可对主元的重要性进行排序，并根据需要略去部分维数，达到降维从而简化模型或对数据进行压缩的效果，采样密度增大。
    • 当数据受到特征影响时，较小特征值所对应的特征向量往往与噪声有关。略去部分维数能起到一定的去噪效果。
    • 完全无参数限制，在计算过程中不需要人为设定参数或是根据任何经验模型对计算进行干预，最终结果只与数据相关。
  • PCA的局限性
    • 假设模型是线性的，也就决定了它能进行的主元分析之间的关系也是线性的。假设数据具有较高信噪比（信息比较多，噪声比较少），具有最高方差的一维向量被看作是主元，而方差较小的变化被认为是噪声。这样被略去的维度对信息的损失较大
    • 假设概率分布模型是指数型(如下图)，没有很好的办法进行分析

PCA和LDA的区别

  • PCA追求降维后能够最大化保持数据内在信息，并通过衡量在投影方向上的数据方差来判断其重要性。但这对数据的区分作用并不大，反而可能使得数据点混杂在一起。 (只用到方差，没用到标签)
  • LDA所追求的目标与PCA不同，不是希望保持数据最多的信息，而是希望数据在降维后能够很容易地被区分开。（用到了标签）

核主成分分析（Kernal PCA)

将主成分分析的线性假设一般化使之适应非线性数据

传统PCA: \(D\) 维样本 \(\left\{\mathbf{x}_n\right\}, n=1,2, \ldots, N, \sum_n \mathbf{x}_n=0\)(去中心化后)

核PCA: 非线性映射 \(\phi(\mathbf{x}), \mathbf{x}_n \longmapsto \phi\left(\mathbf{x}_n\right), \Sigma_n \phi\left(\mathbf{x}_n\right)=0\) \[ \mathrm{C}=\frac{1}{N} \sum_{n=1}^N \phi\left(\mathbf{x}_n\right) \phi\left(\mathbf{x}_n\right)^{\mathrm{T}} \quad \mathrm{Cv}_i=\lambda_i \mathbf{v}_i \] \(C: N\) 个矩阵相加求平均，高维度且 \(\phi\) 末知, 难以直接计算 \(C\) 的特征值与特征向量。考虑引入核函数: \[ k\left(\mathbf{x}_n, \mathbf{x}_m\right)=\phi\left(\mathbf{x}_n\right)^{\mathrm{T}} \phi\left(\mathbf{x}_m\right) \] 核主成分分析 \(\stackrel{v \text { 归一化 }}{\longrightarrow} v=\frac{1}{\|X \mu\|} X \mu=\frac{1}{\sqrt{\mu^T X^T X \mu}} X \mu=\frac{1}{\sqrt{\mu^T \lambda \mu}} X \mu=\frac{1}{\sqrt{\lambda}} X \mu\)

但我们通常仅关注于投影到 \(v\) 后的坐标（即那些特征值），对于新样本 \(x\) : \[ v^T \varphi(x)=\frac{\mathbb{1}}{\sqrt{\lambda}} \boldsymbol{\mu}^T X^T \varphi(x)=\frac{\mathbb{1}}{\sqrt{\lambda}} \mu^T\left[\begin{array}{c} \varphi\left(x_1\right)^T \\ \ldots \\ \varphi\left(x_N\right)^T \end{array}\right] \varphi(x)=\frac{\mathbb{1}}{\sqrt{\lambda}} \boldsymbol{\mu}^T\left[\begin{array}{c} \boldsymbol{k}\left(\boldsymbol{x}_1, x\right) \\ \ldots \\ \boldsymbol{k}\left(\boldsymbol{x}_N, x\right) \end{array}\right] \]

多维尺度分析 MDS (Multi-dimensional Scaling)

MDS基本出发点：将样本之间的距离关系或不相似关系在低维空间中表示出来。

  • 度量型（定量度量样本间距离）
  • 非度量型（定性度量样本间距离）
  • 古典尺度法：主坐标分析方法。根据两两点之间的距离关系（距离矩阵），确定在空间中的坐标。

定义：距离矩阵

一个 \(n \times n\) 矩阵 \(D=\left\{\mathrm{d}_{\mathrm{ij}}\right\}\)被称为距离矩阵, 如果满足以下条件: \[ d_{i j} \geq 0, d_{i i}=0, i, j=1, \ldots, n \] 则\(d_{i j}\) 称为第 \(i\) 与第 \(j\) 个点间的距离

距离矩阵通常不限于欧式距离。

有了距离矩阵, 多维尺度法的目的就是要确定维数 \(k\), 并在 \(k\) 维空间 \(R_k\) 中求 \(n\) 个点 \(\left\{\mathrm{x}_1, \mathrm{x}_2, \mathrm{x}_3, \ldots, \mathrm{x}_{\mathrm{n}}\right\}\), 使得 \(n\) 个点(在原空间）的欧式距离与（降维空间下）距离阵中的相应值在某种意义下尽量接近。

（这里的解释有待商榷）

原始数据矩阵为样本 \(X\) 的 \(n \times q\) 维矩阵, \(n\) 为样本数, \(q\) 是原始数据的维度 (样本表示成行向量)

注意这里的样本矩阵的列是维度，行是样本。和前面PCA部分的矩阵是转置关系。

为了方便计算, 对原始数据矩阵进行中心化: \[ \sum_{i=1}^n x_{i k}=0, k=0,1, . ., q \] 我们首先构造一个内积矩阵 \(B=X X^T, B\) 为 \(n \times n\) 的矩阵。 \[ B=\left[\begin{array}{cccc}x_{11} & x_{12} & \ldots & x_{1 q} \\ \cdot & \cdot & \cdot & \cdot \\ \cdot & . & \cdot & \cdot \\ x_{n 1} & x_{n 2} & \cdot & x_{n q}\end{array}\right] \times\left[\begin{array}{cccc}x_{11} & . & . & x_{n 1} \\ x_{12} & \cdot & . & x_{n 2} \\ \cdot & \cdot & . & \cdot \\ x_{1 q} & . & . & x_{n q}\end{array}\right] \\ =\left[\begin{array}{cccc} x_{11}^2+x_{12}^2+\ldots+x_{1 q}^2 & . & \cdot & x_{11} x_{n 1}+x_{12} x_{n 2}+\ldots+x_{1 q} x_{n q} \\ . & . & . & . \\ \cdot & . & . & . \\ x_{n 1} x_{11}+x_{n 2} x_{12}+\ldots+x_{n q} x_{1 q} & . & . & x_{n 1}^2+x_{n 2}^2+\ldots+x_{n q}^2 \end{array}\right]_{\mathrm{n} \times n} \] \(b_{i j}=\sum_{k=1}^q x_{i k} x_{\mathrm{j} k}=x_i x_j\)

实际应该写为\(x_i x_j^T\)，这里简化的用一维向量来表示，并不影响推导结果

距离矩阵 \(D\) 的计算公式为: \[ d_{i j}^2=\left(x_i-x_j\right)^2=x_i^2+x_j^2-2 x_i x_j \] 因为 \(b_{i j}=x_i x_j\),

所以: \(d_{i j}^2=\sum_{k=1}^q x_{i k}^2+\sum_{k=1}^q x_{j k}^2-2 b_{i j}=b_{i i}+b_{j j}-2 b_{i j}\)

由于进行了中心化处理, 所以: \[ \sum_{i=1}^n x_{i k}=0, k=0,1, . ., q \] 因此有 \[ \begin{array}{ll} \sum_{j=1}^n b_{i j}&=\sum_{j=1}^n \sum_{k=1}^q x_{i k} x_{j k} \\ &=\sum_{k=1}^q\left(x_{i k} \sum_{j=1}^n x_{j k}\right) \\ &=0 \end{array} \]

\[ \begin{array}{ll} \sum_{i=1}^n b_{i j} &=\sum_{i=1}^n \sum_{k=1}^q x_{i k} x_{j k} \\ &=\sum_{k=1}^q\left(x_{j k} \sum_{i=1}^n x_{i k}\right) \\ &=0 \end{array} \]

那么: \[ \begin{aligned} \sum_{i=1}^n d_{i j}^2&=\sum_{i=1}^n b_{i i}+\sum_{i=1}^n b_{j j}-2 \sum_{i=1}^n b_{i j} \\ &=\sum_{i=1}^n b_{i i}+n b_{j j} \\ \sum_{j=1}^n d_{i j}^2&=\sum_{j=1}^n b_{i i}+\sum_{j=1}^n b_{j j}-2 \sum_{j=1}^n b_{i j} \\ &=\sum_{j=1}^n b_{j j}+n b_{i i} \\ \sum_{i=1}^n \sum_{j=1}^n d_{i j}^2&=\sum_{i=1}^n \sum_{j=1}^n b_{j j}+\sum_{i=1}^n n b_{i i}\\ &=n \sum_{j=1}^n b_{j j}+n \sum_{i=1}^n b_{i i}\\ &=2 n T \end{aligned} \]

其中 \(\text { (Trace) }T=\sum_{i=1}^n b_{i i}=\sum_{j=1}^n b_{j j}\)

于是联立以下等式: \[ \begin{aligned} &d_{i j}^2=\sum_{k=1}^q x_{i k}^2+\sum_{k=1}^q x_{j k}^2-2 b_{i j}=b_{i i}+b_{i j}-2 b_{i j} \\ &\sum_{i=1}^n d_{i j}^2=T+n b_{j j} \Rightarrow b_{j j}=\frac{T}{n}+\frac{1}{n} \sum_{i=1}^n d_{i j}^2 \\ &\sum_{j=1}^n d_{i j}^2=T+n b_{i i} \Rightarrow b_{i i}=\frac{T}{n}+\frac{1}{n} \sum_{i=1}^n d_{i j}^2 \\ &\sum_{i=1}^n \sum_{j=1}^n d_{i j}^2=2 n T \Rightarrow \frac{2 T}{n}=\frac{1}{n^2} \sum_{i=1}^n \sum_{j=1}^n d_{i j}^2 \end{aligned} \]

\[ \begin{aligned} &b_{i j}=-\frac{1}{2}\left(d_{i j}^2-b_{i i}-b_{i j}\right) \\ &=-\frac{1}{2}\left(d_{i j}^2-\frac{1}{n} \sum_{j=1}^n d_{i j}^2-\frac{1}{n} \sum_{i=1}^n d_{i j}^2+\frac{2 T}{n}\right) \\ &=-\frac{1}{2}\left(d_{i j}^2-\frac{1}{n} \sum_{j=1}^n d_{i j}^2-\frac{1}{n} \sum_{i=1}^n d_{i j}^2+\frac{1}{n^2} \sum_{i=1}^n \sum_{j=1}^n d_{i j}^2\right) \end{aligned} \] 由上述公式, \(B=X X^T\) 内积矩阵可以完全由 \(D\) 距离阵得出!

以下的步骤类似主成分分析。MDS的主要作用是将距离矩阵变换成内积矩阵

\(\boldsymbol{B}=\frac{\boldsymbol{- 1}}{\mathbf{2}} \boldsymbol{H S H}\), 其中 \(H=I_n-\frac{1}{N} 11^T \quad S(i, j)=D_G^2(i, j)\)

\(B=X X^T\), 可以对角化: \[ B=V \Lambda V^{-1}=V \Lambda^{1 / 2} \Lambda^{1 / 2} V^{-1}=X X^T \] 其中, \(\Lambda\) 是由矩阵 \(B\) 的特征值组成的矩阵, \(V\) 是由特征向量组成的矩阵。

因此如果想通过距离阵还原原始数据, 就选择 \(B\) 矩阵的 \(q\) 个向量, 如果想达到降维的目的, 就选择 \(p(p<q)\) 个特征向量。最终降维的数据 \(X\) 为: \[ \mathrm{X}=V \Lambda^{1 / 2} \]

等距映射 ISO-Metric Mapping

保持数据点内在几何性质(测地距离)

高维空间中直接计算直线距离具有误导性，因为该直线距离在低维嵌入中具有不可达性。

思想：通过局部距离来定义非线性距离度量，在样本较密集的情况下可以实现复杂的非线性距离度量。

对于给定数据 \(X=\left\{x_1, x_2, \ldots, x_N\right\}\), 构造加权图 \(G=\{V, E\}\) \(V\) 是顶点集合, \(E\) 是边的集合

若 $d(i, j)=(x_i, x_j) $ 小于某个值 \(\epsilon(\epsilon-\mathrm{ISOMAP})\)或 \(j\) 是 \(i\) 的 \(K\) 近邻 \((K-\) ISOMAP) ，则顶点 \(i\) 与 \(j\) 的边权值设为 \(d(i, j)\) ，否则为 0

计算图 \(G=\{V, E\}\) 中任意两点间的最短距离，得到矩阵 \(D_G(i, j)\)

  • Dijkstra最短路径算法
  • Floyd-Warshall算法

根据样本间的距离矩阵, 可用MDS（多维尺度法）映射到低维空间, 具体做法（以古典尺度法为例）：

定义平方距离矩阵 \(S(i, j)=D_G^2(i, j)\)，并令 \(H=I_n-\frac{1}{N} 11^T\) (中心矩阵, Centering Matrix)

求矩阵 \(B=\frac{-1}{2} H S H\) 的特征值与特征向量(按特征值降序排列), \(\lambda_p\) 为第 \(p\) 个特征值, \(v_p\) 为对应的特征向量。

降维矩阵为 \(\left[\sqrt{\lambda_1} v_1, \ldots,\left.\sqrt{\lambda_d} v_d\right]_{N \times d}\right.\)

计算步骤

  • 构造临近关系图 对每一个点，将它与指定半径邻域内所有点相连(或与指定个数最近 邻相连)
  • 计算最短路径 计算临近关系图所有点对之间的最短路径，得到距离矩阵
  • 多尺度分析 将高维空间中的数据点投影到低维空间，使投影前后的距离矩阵相似 度最大

优缺点

  • 优点
    • 非线性
    • 非迭代
    • 全局最优
    • 参数可调节
  • 缺点
    • 容易受噪声干扰
    • 在大曲率区域存在短路现象
    • 不适用于非凸参数空间
    • 大样本训练速度慢

局部线性嵌入

保持数据点的原有流形结构：多个点之间的局部线性关系

计算权值, 首先构造局部协方差矩阵 \(C^i\) \[ C_{j k}^i=\left(x_i-x_j\right) \cdot\left(x_i-x_k\right) \] 然后，求解 \[ \underset{w}{\min} \sum_i\left|x_i-\sum_j w_{i j} x_{i j}\right|^2, s.t. \sum_j w_{i j}=1 \]

这里求解\(w_{i,j}\)的含义为，找出一组能最好体现原有流形结构的关系权重

最后可求得闭式解: \[ w_{i j}=\frac{\sum_k\left(C_{j k}^i\right)^{-1}}{\sum_{j k}\left(C_{j k}^i\right)^{-1}} \] 计算低维数据, 求解 \[ \underset{y}{\min} \sum_i\left|y_i-\sum_j w_{i j} y_{i j}\right|^2 \] 通过拉格朗日乘子法可以将其转换为可转化为最小化 \(\min \sum_{i j} M_{i j}\left(y_i \cdot y_j\right)\)

其中, \(M=(I-W)^T(I-W)\)

求解 \(M Y=\lambda Y\)

取 \(Y\) 为 \(M\) 的最小 \(d\) 个非零特征值所对应的特征向量构成的矩阵，最终的输出结果即为 \(N \times d\) 大小的矩阵\(Y\)
